# JSP_SSM_medicine_management
JSP基于SSM医药信息管理系统毕业源码案例设计

## 功能实现：
  管理员角色包含以下功能：登录,信息查询,信息录入,信息删除,信息修改,信息浏览,打印报表,用户管理,权限管理,系统帮助等功能。
  用户角色包含以下功能：管理员只设置了查询功能,一些说明等功能。
  用了技术框架： HTML+CSS+JavaScript+jsp+mysql+Spring+mybatis
